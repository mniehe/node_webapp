#Node Webapp Starter

###*Everything your need to start building a webapp quickly based on Node.JS*
---

####Contains
- Node.JS
- Express.io
- Sass
- Susy
- Compass
- Grunt

####Usage

#####During development use:
```
grunt watch
```
#####For production quality compiling:
```
grunt build
```
#####To run the test server
```
node app/app.js
```
